# Código de Conducta

Como participantes de este proyecto, nos comprometemos a crear un ambiente acogedor, diverso y respetuoso para todos.

## Reglas

- Se respetuoso con los demás.
- No se permite lenguaje ofensivo o discriminatorio.
- Acepta las críticas constructivas.
- Colabora con una mentalidad abierta.

## Contacto

Si observas alguna violación al código de conducta, por favor contacta al administrador del repositorio.
